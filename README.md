# iNtellecta - Unleash Your Inner Intellect


## Overview  
iNtellecta offers students a unified study dashboard with Pomodoro timer, flashcards, videos, documents, and optional background music — all within a sleek, responsive React interface.

---

## Features   
- Pomodoro timer and task manager  
- Flashcard creation, flip & delete  
- Video & Document tabs to access course materials  
- Background music player for focused sessions  
- Dark/Light mode toggle  

---

## Technologies Used  
- **React** – UI component structure, state management  
- **JavaScript** – Core logic and interactivity  
- **HTML & CSS (Tailwind)** – Markup and styling  
- **Vite** – Fast development workflow and build tool  
- **Supabase** – Backend service for authentication, database, and storage  

---

## Installation  
```bash
git clone https://github.com/TommyBryan/iNtellecta.git  
cd iNtellecta  
npm install  
npm run dev  
```
